# ChineseSegment
A 3 layer CNN for Chinese segmentation
## environment
Tesorflow-gpu

Python 3.6

## data
Bakeoff 2005

[Microsoft Research](http://sighan.cs.uchicago.edu/bakeoff2005/)

## train

python segment.py train

We have provided a pre-trained model of 60 epoch.

## test

python segment.py test
